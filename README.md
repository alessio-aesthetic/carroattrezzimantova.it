# carroattrezzimantova.it

Sito statico Next.js per servizio di carroattrezzi a Mantova.

Base grafica: template Tailwind Plus v4, con struttura e componenti adattati per un sito locale ad alta conversione.

## Comandi

```bash
npm install
npm run build
```

Output statico in `out/`, pronto per GitHub Pages.
